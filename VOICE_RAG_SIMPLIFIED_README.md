# 语音RAG服务 - 简化版本 v2.0

## 🎯 **架构简化说明**

基于您现有的完善意图识别系统，我们简化了语音RAG服务，移除重复的意图识别逻辑，直接集成`intelligent_agent.py`进行统一处理。

## 🏗️ **新架构设计**

### 原架构（复杂）
```
语音输入 → 音频转录 → 意图分类 → 服务路由 → RAG/Activity服务 → 语音输出
          ↓          ↓         ↓
      Whisper   自定义分类   重复逻辑
```

### 新架构（简化）
```
语音输入 → 音频转录 → intelligent_agent → 语音输出
          ↓              ↓
      Whisper      统一意图识别
                   ↓
              自动工具调用
              ↓
         RAG/Activity/MCP工具
```

## 🔄 **主要变化**

### 移除的组件
- ❌ `IntentClassifier` 类
- ❌ `TIME_QUERY_KEYWORDS` 和 `ACTIVITY_KEYWORDS`
- ❌ `RAGQueryClient` 类
- ❌ 重复的意图识别逻辑

### 新增的组件
- ✅ `IntelligentQueryClient` 类
- ✅ 集成 `intelligent_agent.py`
- ✅ 智能回退机制

### 保留的组件
- ✅ `AudioProcessor` - 语音转录
- ✅ `TTSProcessor` - 语音合成
- ✅ `PerformanceOptimizer` - 性能优化
- ✅ 所有API端点

## 🚀 **优势**

1. **消除重复**：不再有重复的意图识别逻辑
2. **统一管理**：所有意图识别由`intelligent_agent`统一处理
3. **更强功能**：支持MCP工具调用、搜索、推理等
4. **智能回退**：如果智能代理失败，自动回退到RAG服务
5. **更好维护**：减少代码重复，降低维护成本
